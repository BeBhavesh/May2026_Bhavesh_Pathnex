What is IP Address?

An IP Address is a unique identifier assigned to a device on a network.

🔹 Types of IP
1. Public IP
Assigned by ISP
Accessible over internet
Example: EC2 public IP

2. Private IP
Used inside local network
Not accessible from internet
Example ranges:
192.168.x.x
10.x.x.x
172.16–31.x.x

3. Static IP
Fixed IP address
Used for servers
4. Dynamic IP
Changes automatically (DHCP)
Used in home networks


5. IPv4 vs IPv6
Feature	    IPv4	          IPv6
Size	    32-bit	         128-bit
Example	    192.168.1.1	     2001:db8::1
Limitation	 Limited	      Huge
